API Enpoints

Authentication

POST /api/auth

Handles user authentication. Users must authenticate to access protected routes

Request Body

json
{

 email: string

 password: string

}

Responses

Returns a JSON Web Token that must be included in the `Authorization` header of subsequent requests to protected routes

GET /api/navbar

Retrieves all navigation items

Responses

  • 200 OK: Successfully retrieved all navigation items

POST /api/navbar

Creates a new navigation item

json
{

 "name": "navigation item item name",

 "ref": "navigation item href link",

}
  • 201: Navigation item successfully created

GET /api/maintitle

Retrieves the main title for the portfolio

Responses

  • 200 OK: Successfully retrieved the main title

POST /api/maintitle

Creates a new main title if it doesn't exist

json
{

 "firstName": "FirstName",

 "lastName": "Lastname",

 "position": "Developer",

 "type": "Full Stack"

}
  • 201: Main title successfully created
